# tic_tac_toe
두 명이 번갈아가며 O와 X를 3×3 판에 써서 같은 글자를 가로, 세로, 혹은 대각선 상에 놓이도록 하는 게임 🎮

### Development environment
- react.js
- node.js



### Features
- O와 X, 3×3 게임
- 다음 player 표기
- 뒤로가기 버튼
